# 飞书多维表格任务看板设计

## 表格结构

建议创建一个多维表格，包含以下字段：

| 字段名 | 类型 | 说明 |
|---|---|---|
| 任务名称 | 文本 | 具体任务描述 |
| 阶段 | 单选 | Phase 1/2/3/4 |
| 状态 | 单选 | 待开始/进行中/已完成/阻塞 |
| 优先级 | 单选 | P0/P1/P2/P3 |
| 负责人 | 人员 | 你 or 我 |
| 截止日期 | 日期 | 预计完成时间 |
| 备注 | 文本 | 补充说明 |
